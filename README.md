## 🚀 Deployment

### Option 1 — GitHub Pages (Free, recommended)
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io/ajc-contracting`
5. Add your custom domain in Pages settings

### Option 2 — Netlify (Free, custom domain)
1. Connect this GitHub repo to Netlify
2. Deploy settings: build command = none, publish directory = `/`
3. Add custom domain in Netlify dashboard

### Option 3 — Any web host
Upload all files to your hosting provider's `public_html` folder via FTP.

## 🔍 SEO

The site is fully optimised for Lebanese search traffic:

- **Meta tags** — title, description, keywords in EN/AR/FR on all pages
- **Schema markup** — `GeneralContractor` + `Organization` JSON-LD on homepage
- **Service schemas** — `Service` + `FAQPage` + `BreadcrumbList` JSON-LD on each service page
- **Geo tags** — `LB-JL` / Metn, Mount Lebanon
- **hreflang** — EN, AR, FR language signals on all pages
- **Open Graph & Twitter Cards** — on all pages
- **Sitemap** — `sitemap.xml` covering all pages
- **Robots.txt** — allows all except `/social/`
- **Preconnect** — `fonts.gstatic.com` + DNS-prefetch for Unsplash
- **Font preload** — Barlow Condensed 900 weight
- **Google Analytics GA4** — placeholder on all pages (replace `G-XXXXXXXXXX`)

### Submit to Google Search Console
1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Add your domain
3. Submit `https://www.ajc-contracting.com/sitemap.xml`
